# 🤖 ShimBot-D (심봇드)

> **Discord를 이용한 기능 확장형 비서 봇**

## 🛠️ 기술 스택 (Tech Stack)

### **언어 및 라이브러리** 
1. Go (Golang 1.18+), `github.com/bwmarrin/discordgo`
2. Python, `yt-dlp`

### 사전 필수 환경
* `go`, `FFmpeg`, `Python`

## 🫡 `ShimBot-D`의 개발 이유
- 개발자 Shim의 좌뇌 역할을 대리 수행합니다.
- 어떤 콘텐츠이든지 **공유 → 디스코드 → ShimBot-D**를 클릭하고 보내기를 누른다면, 필요한 작업을 알아서 진행합니다.
- 현재로서는 Youtube 개별 영상 링크를 받으면 Youtube를 다운로드해주며
- Youtube 플레이리스트 및 개별영상링크가 포함된 평문을 받으면 정확히 유튜브 영상만 고른 후 다운로드 메뉴가 나옵니다.
- 다운로드시 `480p`, `720p`, `1080p`, mp3의 옵션을 선택할 수 있습니다.

## 개발 진행 방향
- 모듈화가 간편하도록 utils/base.go에서 모든 핸들러를 등록하도록 설계했습니다.
- utils 내부에 기능을 추가합니다.

---

## ✨ 주요 기능 (Key Features)

* **📥 유튜브 다운:** 유튜브 영상 URL(또는 화질 옵션 `[+resolution]`)을 봇에게 DM으로 보내면 `yt-dlp`로 영상을 다운로드하여 파일 서버로 영상을 제공합니다.
* **📋 장문 텍스트 내 링크 추출 및 목록화:** 유튜브 및 재생목록(Playlist) URL이 포함된 평문 텍스트를 DM으로 보내면, 재생목록 내 개별 영상 링크까지 완벽하게 추출하여 페이지 단위의 목록으로 만들어 줍니다. 유저는 인터랙션 UI를 통해 원하는 항목을 다운로드할 수 있습니다.
* **🧩 유연한 모듈화 설계:** 봇의 기능들이 독립된 모듈 구조로 나누어져 있어, 향후 유튜브 다운로드 외에 새로운 명령어 캐싱이나 기능 확장/유지보수가 매우 수월합니다.
* **🛠️ 인프라 자동 구축 (Zero-Setup Venv):** 서버에 `ffmpeg`와 `python`만 설치되어 있다면, 봇이 실행될 때 스스로 Python 가상환경(`.venv`)을 생성하고 내부에 필요한 종속성(`yt-dlp` 등)을 알아서 셋업합니다.

## ⚙️ 설치 및 구동 방법 (Configuration)

-Linux, Unix 등의 환경
  * `go build -o ShimBot-D`

-Windows
  * `go build -o ShimBot-D.exe`

`pm2 start ShimBot-D` (서버 관리 모듈로 pm2를 채택했습니다. 단순히 ShimBot-D를 실행시켜도 됩니다.`

최초 실행 시 생성되는 `config.json` 파일에 서버 구동에 필요한 정보를 입력해주세요.

* **도메인 (Domain):** 다운로드 완료 후 파일 링크 제공 등에 사용될 도메인 주소
* **포트 (Port):** 봇 웹서버 또는 파일 제공용 네트워크 포트
* **디스코드 봇 토큰 (Discord Bot Token):** 디스코드 개발자 포털에서 발급받은 봇 토큰
